**Chemistry laboratory report LaTeX template — title block, abstract, reagents table, mhchem reaction equations, procedure, results with percent-yield calculations, discussion and conclusion.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/lab-documents/chemistry-lab-report)**, with real-time collaboration and one-second compiles.

![Chemistry Lab Report preview](preview.png)

## Features
- Title block + abstract
- Reagents table with formulas & hazards
- mhchem reaction equations
- Numbered procedure + results
- Percent-yield calculations + discussion

## Use it online (recommended)
Open **[Chemistry Lab Report on LetX »](https://letx.app/templates/lab-documents/chemistry-lab-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/chemistry-lab-report.git
cd chemistry-lab-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
