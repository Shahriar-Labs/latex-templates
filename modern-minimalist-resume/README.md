**Clean single-column modern minimalist resume LaTeX template — Lato sans, one accent color, thin section rules, right-aligned dates. One page, ATS-readable.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/cvs-resumes/modern-minimalist-resume)**, with real-time collaboration and one-second compiles.

![Modern Minimalist Resume preview](preview.png)

## Features
- Clean single-column layout with generous whitespace
- Lato sans-serif + one restrained accent color
- Thin accent section rules; right-aligned dates
- FontAwesome contact icons; clickable links
- One page, ATS-readable, easy to recolor

## Use it online (recommended)
Open **[Modern Minimalist Resume on LetX »](https://letx.app/templates/cvs-resumes/modern-minimalist-resume)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/modern-minimalist-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
