**Elegant wedding invitation LaTeX template — delicate double border with corner ornaments, calligraphic couple names, the invitation wording, date, venue and RSVP line. Soft, romantic palette.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/wedding-invitation)**, with real-time collaboration and one-second compiles.

![Wedding Invitation preview](preview.png)

## Features
- Delicate double border + corner ornaments
- Calligraphic couple names
- Invitation wording, date, venue
- RSVP + reception line
- Soft romantic palette; one page

## Use it online (recommended)
Open **[Wedding Invitation on LetX »](https://letx.app/templates/miscellaneous/wedding-invitation)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/wedding-invitation
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
