## Use it online (recommended)
Open **[Press Release on LetX »](https://letx.app/templates/miscellaneous/press-release)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/press-release.git
cd press-release
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
