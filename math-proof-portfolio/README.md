**Mathematics proof portfolio LaTeX template — color-coded tcolorbox theorem environments (Theorem, Lemma, Proposition, Definition, Corollary) with a QED-boxed Proof environment and several classical results fully proved.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/math-proof-portfolio)**, with real-time collaboration and one-second compiles.

![Math Proof Portfolio preview](preview.png)

## Features
- amsthm + color-coded tcolorbox theorem boxes
- Theorem / Lemma / Proposition / Definition / Corollary
- Proof environment with a QED box
- Several classical results, fully proved
- Clean title block; running header

## Use it online (recommended)
Open **[Math Proof Portfolio on LetX »](https://letx.app/templates/assignments/math-proof-portfolio)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/math-proof-portfolio
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
