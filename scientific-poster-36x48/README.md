## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/scientific-poster-36x48.git
cd scientific-poster-36x48
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
