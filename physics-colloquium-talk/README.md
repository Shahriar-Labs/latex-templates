Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/conferences/physics-colloquium-talk)**, with real-time collaboration and one-second compiles.

![Astrophysics Beamer Talk preview](preview.png)

## Features
- Modern, elegant **Beamer** talk — clean custom theme (no metropolis/fontspec)
- **pdflatex-safe** — compiles anywhere, no XeLaTeX/LuaLaTeX needed
- Title, outline, section, equation, figure, and results-table slides
- Self-contained pgfplots figure and a `booktabs` results table
- Realistic gravitational-wave / Hubble-tension sample talk

## Use it online (recommended)
Open **[Astrophysics Beamer Talk on LetX »](https://letx.app/templates/conferences/physics-colloquium-talk)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/physics-colloquium-talk
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
