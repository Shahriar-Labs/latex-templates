## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/jsps-grant-proposal.git
cd jsps-grant-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
