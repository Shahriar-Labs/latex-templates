**AASTeX 6.3.1 astronomy paper template (ApJ / AJ / AAS journals) — two-column, author affiliations, a deluxetable, equations, a figure, and a bibliography. For astrophysics manuscripts.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/aastex-apj-paper)**, with real-time collaboration and one-second compiles.

![AASTeX Astronomy Paper preview](preview.png)

## Features
- AAS **AASTeX 6.3.1** class — for *ApJ*, *AJ*, *ApJL* and other AAS journals
- Two-column style with multiple authors + affiliations
- Signature **`deluxetable`**, displayed equations, and a pgfplots figure
- Inline bibliography with `\citet`/`\citep`
- Realistic exoplanet transit-photometry sample content

## Use it online (recommended)
Open **[AASTeX Astronomy Paper on LetX »](https://letx.app/templates/journal-articles/aastex-apj-paper)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/aastex-apj-paper.git
cd aastex-apj-paper
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
