**Academic journal cover letter LaTeX template — author letterhead with ORCID, editor address block, manuscript metadata, and a structured body covering significance, scope fit, originality and no-conflict statements.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/journal-cover-letter)**, with real-time collaboration and one-second compiles.

![Journal Cover Letter preview](preview.png)

## Features
- Author letterhead with ORCID
- Editor / journal address block
- Manuscript metadata (title, type, authors)
- Significance, scope-fit, originality statements
- No-conflict + approval statements; one page

## Use it online (recommended)
Open **[Journal Cover Letter on LetX »](https://letx.app/templates/journal-articles/journal-cover-letter)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/journal-cover-letter.git
cd journal-cover-letter
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
