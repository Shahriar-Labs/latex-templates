Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/cvs-resumes/nurse-medical-resume)**, with real-time collaboration and one-second compiles.

## Features
- Credentials header (RN, BSN) + clinical teal accent
- Professional Summary
- Licensure & Certifications (RN, BLS, ACLS…)
- Clinical Experience + Clinical Skills
- One page, ATS-friendly

## Use it online (recommended)
Open **[Nurse / Medical Resume on LetX »](https://letx.app/templates/cvs-resumes/nurse-medical-resume)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/nurse-medical-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
