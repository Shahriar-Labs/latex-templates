## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/multiple-choice-test.git
cd multiple-choice-test
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
