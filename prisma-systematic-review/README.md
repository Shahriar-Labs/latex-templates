**Systematic review & meta-analysis LaTeX template — structured abstract, eligibility criteria, a TikZ PRISMA 2020 flow diagram with record counts and exclusion boxes, study-characteristics table and results.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/prisma-systematic-review)**, with real-time collaboration and one-second compiles.

![PRISMA Systematic Review preview](preview.png)

## Features
- Structured abstract + PRISMA 2020 methods
- TikZ PRISMA flow diagram (counts + exclusions)
- Eligibility criteria + search strategy
- Study-characteristics table (booktabs)
- Realistic meta-analysis sample

## Use it online (recommended)
Open **[PRISMA Systematic Review on LetX »](https://letx.app/templates/journal-articles/prisma-systematic-review)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/prisma-systematic-review
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
