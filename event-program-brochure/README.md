- Decorative header (event, date, venue)
- Schedule / agenda
- Featured-speaker cards
- Sponsors / acknowledgements strip
- Event footer; one page

## Use it online (recommended)
Open **[Event Program Brochure on LetX »](https://letx.app/templates/miscellaneous/event-program-brochure)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/event-program-brochure.git
cd event-program-brochure
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
