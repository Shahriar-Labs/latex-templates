## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/api-documentation-swagger.git
cd api-documentation-swagger
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
