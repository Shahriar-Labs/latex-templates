**Laboratory safety manual LaTeX template — cover + document control, PPE and chemical-hazard tables, colored warning callouts, emergency procedures, waste disposal and an emergency-contacts table.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/lab-documents/lab-safety-manual)**, with real-time collaboration and one-second compiles.

![Lab Safety Manual preview](preview.png)

## Features
- Cover + document control box
- PPE + chemical-hazard tables
- Colored warning callouts (tcolorbox)
- Emergency procedures + waste disposal
- Emergency-contacts table

## Use it online (recommended)
Open **[Lab Safety Manual on LetX »](https://letx.app/templates/lab-documents/lab-safety-manual)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/lab-safety-manual.git
cd lab-safety-manual
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
