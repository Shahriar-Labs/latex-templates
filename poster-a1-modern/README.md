**Modern A1 academic conference poster LaTeX template (tikzposter) — bold title band, a multi-column layout with Introduction, Methods, Results (chart + table), Conclusion and References blocks.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/posters/poster-a1-modern)**, with real-time collaboration and one-second compiles.

![Modern Conference Poster (A1) preview](preview.png)

## Features
- A1 portrait tikzposter layout
- Bold title band with authors/affiliations
- Multi-column blocks: intro, methods, results
- Results chart + table
- Conclusion + references

## Use it online (recommended)
Open **[Modern Conference Poster (A1) on LetX »](https://letx.app/templates/posters/poster-a1-modern)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/poster-a1-modern
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
