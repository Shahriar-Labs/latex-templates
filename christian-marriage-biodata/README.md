## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/christian-marriage-biodata.git
cd christian-marriage-biodata
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
