**Data science / machine learning report LaTeX template — title block, executive summary, problem statement, EDA with a pgfplots chart, model-comparison table (accuracy/precision/recall/F1) and conclusions.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/data-science-report)**, with real-time collaboration and one-second compiles.

![Data Science Report preview](preview.png)

## Features
- Title block + executive summary
- Problem statement + feature table
- EDA with a pgfplots chart
- Model-comparison table (acc/prec/recall/F1)
- Discussion + future work; optional code block

## Use it online (recommended)
Open **[Data Science Report on LetX »](https://letx.app/templates/assignments/data-science-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/data-science-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
