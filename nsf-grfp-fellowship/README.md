**NSF Graduate Research Fellowship (GRFP) statements LaTeX template — Personal/Background and Graduate Research Plan statements with Intellectual Merit and Broader Impacts callouts.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/nsf-grfp-fellowship)**, with real-time collaboration and one-second compiles.

![NSF GRFP Fellowship Statements preview](preview.png)

## Features
- Both required GRFP statements, clearly separated
- Personal/Background + Graduate Research Plan
- Intellectual Merit & Broader Impacts callouts
- titlesec headings; running header
- Realistic STEM applicant sample

## Use it online (recommended)
Open **[NSF GRFP Fellowship Statements on LetX »](https://letx.app/templates/grants/nsf-grfp-fellowship)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/nsf-grfp-fellowship.git
cd nsf-grfp-fellowship
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
