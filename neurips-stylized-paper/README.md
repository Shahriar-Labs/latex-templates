## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/neurips-stylized-paper
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
