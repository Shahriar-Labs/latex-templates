Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/teaching-statement)**, with real-time collaboration and one-second compiles.

## Features
- Matching academic title block + boxed philosophy
- Philosophy, Experience, Course Development sections
- Mentoring & Inclusion, Future Goals
- Teaching-portfolio booktabs summary table
- Realistic sample content; ~2 pages

## Use it online (recommended)
Open **[Teaching Statement on LetX »](https://letx.app/templates/academic-career/teaching-statement)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/teaching-statement
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
