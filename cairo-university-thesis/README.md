## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/cairo-university-thesis.git
cd cairo-university-thesis
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
