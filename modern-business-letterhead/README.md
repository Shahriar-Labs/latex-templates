**Modern corporate letterhead + letter LaTeX template — logo placeholder, company name & contact header with an accent side bar, date, recipient, subject line, business body and a registration footer.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/formal-letters/modern-business-letterhead)**, with real-time collaboration and one-second compiles.

![Modern Business Letterhead preview](preview.png)

## Features
- Logo placeholder + company name & contact header
- Accent side bar + strapline
- Date, reference, recipient, subject line
- Professional business letter body
- Registration footer; one page

## Use it online (recommended)
Open **[Modern Business Letterhead on LetX »](https://letx.app/templates/formal-letters/modern-business-letterhead)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/modern-business-letterhead
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
