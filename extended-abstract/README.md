**Conference extended abstract (2-page mini-paper) LaTeX template — title, authors & affiliations, condensed Introduction/Method/Results sections with a chart or table, conclusion and references.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/conferences/extended-abstract)**, with real-time collaboration and one-second compiles.

![Extended Abstract preview](preview.png)

## Features
- Title + authors & affiliations
- Condensed Introduction / Method / Results
- A compact pgfplots chart or table
- Conclusion + references
- ~2 pages

## Use it online (recommended)
Open **[Extended Abstract on LetX »](https://letx.app/templates/conferences/extended-abstract)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/extended-abstract.git
cd extended-abstract
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
