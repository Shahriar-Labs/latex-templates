**Seminar / lecture handout LaTeX template — header, Key Concepts in colored tcolorbox call-outs with formulas, a summary, discussion questions and further reading. Realistic academic sample.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/seminar-handout)**, with real-time collaboration and one-second compiles.

![Seminar Handout preview](preview.png)

## Features
- Header (title, presenter, course, date)
- Key Concepts in colored tcolorbox call-outs
- Formulas + a worked example
- Summary, discussion questions
- Further reading; ~2 pages

## Use it online (recommended)
Open **[Seminar Handout on LetX »](https://letx.app/templates/assignments/seminar-handout)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/seminar-handout.git
cd seminar-handout
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
