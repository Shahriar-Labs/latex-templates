**Design-forward creative studio letterhead LaTeX template — a bold geometric TikZ header band with studio name & tagline, contact strip, letter body and a dark footer. Expressive yet elegant.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/formal-letters/creative-letterhead)**, with real-time collaboration and one-second compiles.

![Creative Studio Letterhead preview](preview.png)

## Features
- Bold geometric TikZ header band
- Studio name + tagline + contact strip
- Expressive accent palette
- Letter body + signature with socials
- Dark footer bar; one page

## Use it online (recommended)
Open **[Creative Studio Letterhead on LetX »](https://letx.app/templates/formal-letters/creative-letterhead)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/creative-letterhead.git
cd creative-letterhead
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
