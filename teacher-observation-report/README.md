## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/teacher-observation-report.git
cd teacher-observation-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
