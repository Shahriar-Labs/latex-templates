**Monthly personal budget planner LaTeX template — a summary box, a savings-goal tracker, and category tables (income, housing, transport, food, utilities, savings) with budgeted/actual/difference columns.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/personal-budget-planner)**, with real-time collaboration and one-second compiles.

![Personal Budget Planner preview](preview.png)

## Features
- Summary box (income/expenses/net)
- Savings-goal progress tracker
- Category tables: housing, transport, food…
- Budgeted / actual / difference columns
- Realistic sample; one page

## Use it online (recommended)
Open **[Personal Budget Planner on LetX »](https://letx.app/templates/miscellaneous/personal-budget-planner)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/personal-budget-planner.git
cd personal-budget-planner
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
