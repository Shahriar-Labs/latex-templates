**Statistical analysis report LaTeX template — title block, boxed abstract, research question, methods, a pgfplots chart, a regression results table with coefficients and p-values, and interpretation.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/statistics-report)**, with real-time collaboration and one-second compiles.

![Statistics Report preview](preview.png)

## Features
- Title block + boxed abstract
- Research question + methods
- pgfplots exploratory chart
- Regression table (coefficients, SE, p-values)
- Results interpretation + conclusion

## Use it online (recommended)
Open **[Statistics Report on LetX »](https://letx.app/templates/assignments/statistics-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/statistics-report.git
cd statistics-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
