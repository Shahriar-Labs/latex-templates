**Elegant landscape certificate of achievement LaTeX template — navy & gold double border with corner ornaments, ribbon badge, serif title, recipient name, citation, and dual signature lines. One page.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/certificate-of-achievement)**, with real-time collaboration and one-second compiles.

![Certificate of Achievement preview](preview.png)

## Features
- Landscape layout with navy & gold double border
- Decorative corner ornaments + ribbon badge
- Elegant serif title and recipient name
- Citation body + dual signature lines and date
- Fully recolorable; one page

## Use it online (recommended)
Open **[Certificate of Achievement on LetX »](https://letx.app/templates/miscellaneous/certificate-of-achievement)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/certificate-of-achievement.git
cd certificate-of-achievement
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
