## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/bill-of-materials-bom.git
cd bill-of-materials-bom
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
