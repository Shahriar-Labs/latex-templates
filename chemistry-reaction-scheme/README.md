**Organic chemistry reaction scheme LaTeX template using chemfig — a single-step reaction with a reagent arrow, a multi-step synthesis, and a reaction mechanism with curved electron-pushing arrows. Fully editable.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/chemistry-reaction-scheme)**, with real-time collaboration and one-second compiles.

![Chemistry Reaction Scheme preview](preview.png)

## Features
- Built with the chemfig package
- Single-step reaction with a reagent arrow
- Multi-step synthesis (benzene → aniline)
- Reaction mechanism with curved electron-pushing arrows
- mhchem formulas; fully editable

## Use it online (recommended)
Open **[Chemistry Reaction Scheme on LetX »](https://letx.app/templates/miscellaneous/chemistry-reaction-scheme)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/chemistry-reaction-scheme.git
cd chemistry-reaction-scheme
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
