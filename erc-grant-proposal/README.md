Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/erc-grant-proposal)**, with real-time collaboration and one-second compiles.

![ERC Grant Proposal preview](preview.png)

## Features
- ERC Part B1 Extended Synopsis structure
- Boxed project metadata + Key Output callout
- Ground-breaking Nature & Objectives
- Methodology + Resources & Feasibility
- PI track record; realistic sample

## Use it online (recommended)
Open **[ERC Grant Proposal on LetX »](https://letx.app/templates/grants/erc-grant-proposal)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/erc-grant-proposal.git
cd erc-grant-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
