## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/medical-history-questionnaire.git
cd medical-history-questionnaire
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
