## Features
- Education-first ordering (GPA, coursework, honors)
- Projects with tech stacks
- Internships / experience + skills
- Activities & leadership
- Clean entry-level layout; one page

## Use it online (recommended)
Open **[College Student Resume on LetX »](https://letx.app/templates/cvs-resumes/college-student-resume)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/college-student-resume.git
cd college-student-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
