## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/wellcome-trust-grant.git
cd wellcome-trust-grant
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
