**Bioinformatics analysis report LaTeX template — abstract, methods with a pipeline/tool-version table, a differential-expression results table, a pgfplots chart, and monospace gene/command formatting. Realistic RNA-seq sample.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/lab-documents/bioinformatics-report)**, with real-time collaboration and one-second compiles.

![Bioinformatics Report preview](preview.png)

## Features
- Abstract + methods with tool-version table
- Differential-expression results table
- pgfplots chart (e.g. volcano/bar)
- Monospace gene/command formatting
- Realistic RNA-seq sample

## Use it online (recommended)
Open **[Bioinformatics Report on LetX »](https://letx.app/templates/lab-documents/bioinformatics-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/bioinformatics-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
