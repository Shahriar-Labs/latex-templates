\begin{posterblock}{Background \& Motivation}
Accurate prediction of protein--ligand binding affinity is a cornerstone of structure-based drug discovery.
Traditional approaches such as molecular docking and free-energy perturbation remain computationally prohibitive for
virtual screening campaigns exceeding $10^6$ compounds. Deep learning methods, particularly those operating on molecular
graphs, offer a compelling alternative, yet existing architectures struggle to capture long-range electrostatic
interactions and solvent effects at protein--ligand interfaces.

We introduce \textbf{GraphBind}, a graph neural network that jointly encodes ligand topology, protein pocket geometry,
and interfacial non-covalent interaction fingerprints within a unified message-passing scheme. GraphBind leverages
hierarchical attention pooling to prioritize pharmacophoric contacts and achieves state-of-the-art accuracy on three
benchmark datasets spanning kinases, proteases, and GPCRs.
\end{posterblock}

\begin{posterblock}{Dataset \& Preprocessing}
We curated \textbf{BindBench-v2}, a non-redundant set of 14\,872 protein--ligand complexes drawn from the PDBbind
refined set (v2024) and augmented with MD-refined decoy structures from the DUD-E+ benchmark.
\begin{itemize}[leftmargin=*, itemsep=6pt, topsep=6pt]
  \item Ligand graphs: nodes = heavy atoms (C, N, O, S, P, halogens); edges = covalent bonds plus
        spatial edges ($d \leq 4.0\,\text{\AA}$, non-covalent).
  \item Protein graphs: nodes = $C_\alpha$ atoms of pocket residues ($\leq 8\,\text{\AA}$ from any ligand atom);
        edges = sequential neighbors ($|i-j| \leq 3$) and spatial proximity ($d_{C_\alpha} \leq 12\,\text{\AA}$).
  \item Interface features: hydrogen-bond donor/acceptor flags, $\pi$--$\pi$ stacking indicators,
        salt-bridge detection, hydrophobic contact area.
  \item Affinity labels: $\mathrm{p}K_d = -\log_{10}(K_d)$; experimental $K_d$ from PDBbind, cross-checked
        against Binding MOAD and ChEMBL~34.
  \item Train/val/test split: 10\,420 / 2\,226 / 2\,226 complexes, stratified by protein family to prevent
        homology leakage.
\end{itemize}
\end{posterblock}

\begin{posterblock}{Model Architecture}
GraphBind employs a three-stream encoder with cross-attention fusion (\textbf{Figure~1}).
\begin{itemize}[leftmargin=*, itemsep=6pt, topsep=6pt]
  \item \textbf{Ligand Encoder:} 6-layer GIN with edge-feature gates; initial node features from RDKit
        (atom type, hybridization, chirality, partial charge).
  \item \textbf{Protein Encoder:} 4-layer E(n)-equivariant GNN operating on $C_\alpha$ coordinates;
        geometric message passing via vector features ensures SE(3) equivariance.
  \item \textbf{Interface Module:} Pairwise attention between ligand and protein node sets;
        distance-aware bias $b_{ij} = -\alpha \cdot |\mathbf{x}_i - \mathbf{x}_j|^2$ with learned $\alpha$.
  \item \textbf{Global Readout:} Set2Set pooling followed by a 3-layer MLP predicting $\mathrm{p}K_d$
        with heteroscedastic uncertainty via a dual-head output ($\mu, \sigma^2$).
\end{itemize}
Total parameters: 4.8M. Trained with AdamW, cosine annealing, batch size 64, for 300 epochs on 4$\times$A100 GPUs.
\end{posterblock}

\begin{posterblock}{Computational Efficiency}
\begin{itemize}[leftmargin=*, itemsep=6pt, topsep=6pt]
  \item Inference time: $< 50$\,ms per complex on a single A100 GPU; scales linearly with pocket residue count.
  \item GraphBind screens 1M compounds against a kinase target in $\sim$14 hours on an 8-GPU node,
        versus $\sim$600K CPU-core-hours for Glide SP docking.
  \item Model checkpoints and inference code released under Apache 2.0 at \texttt{github.com/synapse-inst/graphbind}.
\end{itemize}
\end{posterblock}

\begin{posterblock}{Discussion}
GraphBind's strong performance stems from three design choices. First, the interface attention module explicitly
encodes non-covalent interaction types, enabling the model to distinguish hydrogen bonds from hydrophobic contacts
rather than treating all contacts uniformly. Second, SE(3)-equivariant protein encoding ensures that predictions are
invariant to rigid-body transformations of the pocket, improving generalization to complexes crystallized in different
space groups. Third, heteroscedastic uncertainty quantification allows practitioners to flag low-confidence predictions
during virtual screening triage.

Limitations include reliance on crystallographic pocket definitions (reducing applicability to apo structures) and a
current inability to model covalent or metalloprotein ligands. We are extending the framework to handle induced-fit
effects via docking-guided conformational sampling.
\end{posterblock}

\begin{posterblock}{Conclusions}
\begin{enumerate}[leftmargin=*, itemsep=8pt, topsep=6pt]
  \item GraphBind sets a new state of the art for structure-based binding affinity prediction, with
        $\mathbf{R = 0.862}$ and $\mathbf{RMSE = 0.83}$ on BindBench-v2.
  \item The interface attention module and SE(3)-equivariant protein encoder are both critical,
        contributing $+0.061$ and $+0.030$ Pearson $R$ respectively.
  \item GPU-accelerated inference enables million-scale virtual screening in under one day, positioning
        GraphBind as a practical tool for early-stage hit discovery.
  \item Uncertainty estimates correlate with prediction error (Spearman $\rho = 0.41$), providing a
        calibratable confidence filter for experimental follow-up.
\end{enumerate}
\end{posterblock}

\begin{accentblock}{Future Work}
\begin{itemize}[leftmargin=*, itemsep=6pt, topsep=6pt]
  \item Extend to covalent inhibitors and metalloenzyme targets via reaction-aware edge features.
  \item Integrate with diffusion-based pocket-conditioned ligand generation (co-design with Vertex
        Therapeutics generative chemistry platform).
  \item Active-learning loop connecting GraphBind predictions to automated synthesis and SPR validation
        at the Nimbus High-Throughput Screening Facility.
  \item Multi-task extension to simultaneously predict ADMET endpoints alongside affinity.
\end{itemize}
