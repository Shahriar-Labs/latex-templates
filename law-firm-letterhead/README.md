Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/formal-letters/law-firm-letterhead)**, with real-time collaboration and one-second compiles.

- Classic serif firm name + attorney roster
- Office contact row + delivery line
- Bold Re: matter line
- Formal legal letter body
- Confidentiality footer

## Use it online (recommended)
Open **[Law Firm Letterhead on LetX »](https://letx.app/templates/formal-letters/law-firm-letterhead)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/law-firm-letterhead
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
