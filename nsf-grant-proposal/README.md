Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/nsf-grant-proposal)**, with real-time collaboration and one-second compiles.

## Features
- NSF Project Description structure
- Overview, Intellectual Merit, Broader Impacts
- Research Plan with milestone/timeline table
- Budget-summary table + references
- titlesec headings; realistic CS sample

## Use it online (recommended)
Open **[NSF Grant Proposal on LetX »](https://letx.app/templates/grants/nsf-grant-proposal)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/nsf-grant-proposal.git
cd nsf-grant-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
