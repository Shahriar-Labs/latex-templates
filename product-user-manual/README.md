**Product user manual LaTeX template — cover page, table of contents, safety callouts, getting-started and setup steps, features, a troubleshooting table, specifications and warranty. Uses tcolorbox tips/warnings.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/books/product-user-manual)**, with real-time collaboration and one-second compiles.

![Product User Manual preview](preview.png)

## Features
- Cover page + table of contents
- Safety callouts (tcolorbox warnings)
- Getting-started + numbered setup steps
- Troubleshooting table + specifications
- Warranty; consumer-electronics sample

## Use it online (recommended)
Open **[Product User Manual on LetX »](https://letx.app/templates/books/product-user-manual)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/product-user-manual
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
