**Versatile letter of intent LaTeX template — clean letterhead with accent rule, date, recipient block, subject line, structured multi-paragraph body and signature. Suitable for jobs, grad programs or business.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/formal-letters/letter-of-intent)**, with real-time collaboration and one-second compiles.

![Letter of Intent preview](preview.png)

## Features
- Clean letterhead with a thin accent rule
- Date, recipient block, subject line
- Structured multi-paragraph body
- Signature block
- Versatile: job / grad program / business

## Use it online (recommended)
Open **[Letter of Intent on LetX »](https://letx.app/templates/formal-letters/letter-of-intent)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/letter-of-intent
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
