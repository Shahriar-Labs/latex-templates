**Workout log / training journal LaTeX template — warm-up box, daily-metrics tracker, an exercise table (sets/reps/weight/RPE/notes), a cardio section and a session-notes area.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/workout-log)**, with real-time collaboration and one-second compiles.

![Workout Log preview](preview.png)

## Features
- Warm-up & mobility box
- Daily-metrics tracker (weight/sleep/energy)
- Exercise table (sets/reps/weight/RPE/notes)
- Cardio & conditioning section
- Session notes; realistic sample

## Use it online (recommended)
Open **[Workout Log on LetX »](https://letx.app/templates/miscellaneous/workout-log)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/workout-log.git
cd workout-log
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
