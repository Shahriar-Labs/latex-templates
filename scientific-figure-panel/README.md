**Publication-quality multi-panel scientific figure LaTeX template — a 2x2 grid of labelled (A-D) pgfplots panels (line+error bands, grouped bars with significance, scatter+fit, heatmap) with a shared multi-part caption.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/scientific-figure-panel)**, with real-time collaboration and one-second compiles.

![Scientific Figure Panel preview](preview.png)

## Features
- Publication-quality 2x2 multi-panel figure
- Labelled (A-D) self-contained pgfplots panels
- Line + error bands, grouped bars, scatter+fit, heatmap
- Significance markers + shared multi-part caption
- Robust pgfplots; one page

## Use it online (recommended)
Open **[Scientific Figure Panel on LetX »](https://letx.app/templates/miscellaneous/scientific-figure-panel)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/scientific-figure-panel
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
