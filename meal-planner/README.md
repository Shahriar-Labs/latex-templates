**Weekly meal planner LaTeX template — a 7-day breakfast/lunch/dinner/snacks grid, a categorized grocery checklist and a notes/prep box. Clean, printable, easy to fill in.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/meal-planner)**, with real-time collaboration and one-second compiles.

![Weekly Meal Planner preview](preview.png)

## Features
- 7-day breakfast/lunch/dinner/snacks grid
- Categorized grocery checklist
- Notes & prep box
- Light zebra striping
- Printable; realistic sample

## Use it online (recommended)
Open **[Weekly Meal Planner on LetX »](https://letx.app/templates/miscellaneous/meal-planner)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/meal-planner
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
