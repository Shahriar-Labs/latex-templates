**Postdoctoral application LaTeX template — academic letterhead cover letter expressing fit and summarizing PhD work, plus a short proposed-research section with aims linked to the target lab.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/postdoc-application)**, with real-time collaboration and one-second compiles.

![Postdoc Application preview](preview.png)

## Features
- Academic letterhead cover letter
- Summarizes PhD work + fit to target lab
- Short proposed-research section with aims
- Clean serif academic styling
- Realistic life-sciences/CS sample

## Use it online (recommended)
Open **[Postdoc Application on LetX »](https://letx.app/templates/academic-career/postdoc-application)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/postdoc-application.git
cd postdoc-application
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
