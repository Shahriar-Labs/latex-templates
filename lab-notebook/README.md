**Structured lab notebook / research log LaTeX template — header block (project, researcher, date, entry no.), and per-entry Objective, Materials, Procedure, Observations, Results & Conclusion sections. Includes a filled sample entry.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/lab-documents/lab-notebook)**, with real-time collaboration and one-second compiles.

![Lab Notebook preview](preview.png)

## Features
- Header block: project, researcher, date, entry no.
- Per-entry Objective, Materials, Procedure
- Observations, Results table, Conclusion
- Scientific accent palette; CONFIDENTIAL footer
- Includes a filled realistic sample entry

## Use it online (recommended)
Open **[Lab Notebook on LetX »](https://letx.app/templates/lab-documents/lab-notebook)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/lab-notebook.git
cd lab-notebook
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
