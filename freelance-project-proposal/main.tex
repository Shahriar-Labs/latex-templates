% PROJECT OVERVIEW

\section{Project Overview}
Aetheria Wellness Co. is a premium brand, but its current website lacks the matching visual quality. Slow page loads and non-optimized mobile checkout impact customer acquisition and sales.

\parbreak
Apex Digital Studio proposes to design and develop a high-performance Next.js frontend integrated with Sanity CMS for easy editing and Shopify for inventory management, delivering a fast, responsive, and visually stunning customer experience.

\vspace{0.05in}

% =========================================================================
% OBJECTIVES
% =========================================================================
\section{Objectives}
We aim to achieve the following performance and design goals:
\begin{itemize}
    \item \textbf{Brand Elevation}: Modernize the visual design to reflect a premium wellness brand identity.
    \item \textbf{Conversion Optimization}: Redesign the checkout funnel to improve conversions by 15--20\%.
    \item \textbf{Performance}: Achieve Google Lighthouse Speed scores of 90+ (load time under 1.5s).
    \item \textbf{Content Control}: Provide a headless CMS for independent marketing campaigns.
\end{itemize}

\vspace{0.05in}

% =========================================================================
% SCOPE OF WORK
% =========================================================================
\section{Scope of Work}
The project spans four phases to ensure a seamless design and deployment cycle:
\begin{itemize}
    \item \textbf{Phase 1: Discovery \& Strategy}: Stakeholder workshops, user flows, and mobile-first wireframes.
    \item \textbf{Phase 2: Visual UI Design}: Designing high-fidelity layouts in Figma and establishing a style guide.
    \item \textbf{Phase 3: Development}: Building the Next.js frontend and integrating Sanity CMS and Shopify APIs.
    \item \textbf{Phase 4: QA \& Launch}: Cross-device testing, SEO optimization, and Vercel hosting setup.
\end{itemize}
