**PhD annual progress report LaTeX template — aligned student/supervisor info box, Research Objectives, Progress, a milestones status table, publications, challenges, next-period plan and a supervisor comments box.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/phd-progress-report)**, with real-time collaboration and one-second compiles.

![PhD Progress Report preview](preview.png)

## Features
- Aligned student/supervisor info box
- Research Objectives + Progress This Period
- Milestones & timeline status table
- Publications, challenges, next-period plan
- Supervisor's comments box

## Use it online (recommended)
Open **[PhD Progress Report on LetX »](https://letx.app/templates/academic-career/phd-progress-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/phd-progress-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
