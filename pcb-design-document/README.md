## Use it online (recommended)
Open **[PCB Design Document on LetX »](https://letx.app/templates/engineering/pcb-design-document)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/pcb-design-document.git
cd pcb-design-document
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
