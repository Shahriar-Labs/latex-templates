\begin{tcolorbox}[colback=lightbg, colframe=secondary, title=\textbf{\faInfoCircle\ High-Speed Routing Best Practices}]
\fontsize{9}{11}\selectfont
\begin{itemize}[leftmargin=*]
    \item \textbf{Reference Planes:} High-speed differential pairs must run continuously over an uninterrupted ground reference plane (L2 or L5). Avoid routing across splits or voids.
    \item \textbf{Via Count:} Limit via count on high-speed paths (PCIe, USB 3.2) to a maximum of two. Each via transition must have an adjacent ground return via placed within 15 mils.
    \item \textbf{Length Matching:} USB 3.2 Rx/Tx differential pairs must be length-matched within $\pm$ 5 mils. DDR4 data lanes within the same byte lane must be length-matched within $\pm$ 10 mils.
\end{itemize}
\end{tcolorbox}

% --- Section 6: Design Rule Check (DRC) Checklist ---
\section{Pre-Release Layout Checklist}
Before exporting manufacturing Gerber files, layout design rules must be validated against the manufacturing checklist below.

\begin{enumerate}[label=\lbrack\ \rbrack, leftmargin=*]
    \item Verify that all 50 $\Omega$ single-ended traces match width constraints.
    \item Verify that return vias are placed next to high-speed signal vias.
    \item Confirm PMIC power trace widths satisfy current requirements (minimum 20 mil width per 1A).
    \item Ensure decoupling capacitors are placed directly at the pin entry points.
    \item Run complete DRC check in Altium and verify zero violations.
\end{enumerate}
