**Journal response-to-reviewers LaTeX template — color-coded reviewer comment cards, green author-response bars, and shaded revised-text quotes. Reusable structure for any peer-review rebuttal.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/response-to-reviewers)**, with real-time collaboration and one-second compiles.

![Response to Reviewers preview](preview.png)

## Features
- Color-coded reviewer comment cards (tcolorbox)
- Green author-response bars for clear threading
- Shaded italics blocks for quoted revised text
- Per-reviewer sections; breakable across pages
- Reusable for any peer-review rebuttal

## Use it online (recommended)
Open **[Response to Reviewers on LetX »](https://letx.app/templates/journal-articles/response-to-reviewers)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/response-to-reviewers
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
