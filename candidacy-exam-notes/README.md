**PhD candidacy / qualifying exam study-notes LaTeX template — topic sections with outlines, color-coded tcolorbox key-results/theorem boxes, definitions, exam-tip callouts and formulas via amsmath.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/candidacy-exam-notes)**, with real-time collaboration and one-second compiles.

## Features
- Topic sections with concise outlines
- Color-coded theorem/key-result boxes
- Definitions + amsmath formulas
- Exam-tip callouts
- Compact reference list

## Use it online (recommended)
Open **[Candidacy Exam Notes on LetX »](https://letx.app/templates/assignments/candidacy-exam-notes)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/candidacy-exam-notes.git
cd candidacy-exam-notes
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
