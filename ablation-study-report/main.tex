% ── Abstract ──────────────────────────────────────────────────
{\bfseries\color{accent}Abstract.}\;
We conduct a systematic ablation study of \modelname, an equivariant graph neural network
for molecular property prediction on the \dataset\ benchmark. By sequentially removing
eight architectural components and measuring degradation in \metric\ (\(\metricunit\)),
we identify equivariant message-passing and residual connections as the dominant
contributors to model accuracy. Layer normalization and attention-weighted aggregation
provide moderate gains, while auxiliary denoising and global context injection show
diminishing returns. The full model achieves \metric\ \(= 0.312\), degrading to
\(0.478\) with all components removed. We provide a compute-budget breakdown and a
modular dependency graph to guide future architecture design.

\vspace{8pt}

\section{Experimental Setup}
% ═══════════════════════════════════════════════════════════════

\subsection{Model Architecture}
\modelname\ is an equivariant graph neural network designed for molecular graphs. Each
molecule is represented as a graph \(G = (V, E)\) where nodes \(v \in V\) encode atomic
species and edges \(e \in E\) encode interatomic distances. The architecture comprises
eight modular components:

\begin{itemize}[nosep,leftmargin=*,itemsep=1pt]
  \item \textbf{Equivariant Message-Passing (EquiMP):} SE(3)-equivariant message functions
        that operate on vector-valued features, ensuring rotational invariance of scalar
        predictions.
  \item \textbf{Attention-Weighted Aggregation (Attn):} Multi-head graph attention over
        neighbor messages, learned per message-passing layer.
  \item \textbf{RBF Distance Encoding (RBF):} Radial basis function expansion of
        pairwise distances into 64-dimensional edge embeddings.
  \item \textbf{Residual Connections (Res):} Additive skip connections wrapping each
        message-passing block with pre-activation.
  \item \textbf{Layer Normalization (LN):} Pre-layer normalization applied before each
        message-passing and feed-forward sub-layer.
  \item \textbf{Edge Feature Updating (Edge):} Bidirectional edge feature refinement
        after each message-passing round.
  \item \textbf{Global Context Injection (Global):} A learned global token appended to the
        node set and updated via cross-attention at each layer.
  \item \textbf{Auxiliary Denoising (Aux):} A coordinate-denoising auxiliary loss
        added to the primary regression objective at training time.
\end{itemize}

\subsection{Dataset and Metric}
We evaluate on the \dataset\ dataset (130\,K organic molecules, up to 9 heavy atoms
C/N/O/F). The target is the HOMO-LUMO gap (\(\metricunit\)); we report mean absolute
error (\metric). Training uses 110\,K molecules, validation 10\,K, test 10\,K.

\subsection{Training Configuration}

\begin{tabularx}{\textwidth}{@{}>{\color{accent}\bfseries}l@{\hspace{8mm}}>{\raggedright\arraybackslash}X@{}}
  Optimizer        & AdamW (\(\beta_1 = 0.9\), \(\beta_2 = 0.999\), weight decay \(= 10^{-4}\)) \\
  Learning rate    & \(3 \times 10^{-4}\) with cosine decay to \(10^{-6}\) over 300 epochs \\
  Batch size       & 128 (single Quanta A100-80GB) \\
  Epochs           & 300 (early stopping patience 40 on validation \metric) \\
  Message-passing layers & 6 \\
  Hidden dimension  & 256 \\
  Attention heads  & 8 \\
  RBF basis        & 64 radial bases, cutoff \(r_{\text{max}} = 5.0\)\,\AA \\
  Auxiliary weight  & \(\lambda_{\text{aux}} = 0.1\) \\
\end{tabularx}

\subsection{Dominant Components}
Equivariant message-passing and residual connections account for the largest individual
degradations (\(\Delta = +0.022\) and \(+0.027\) respectively). Removing EquiMP forces
the model to learn rotation invariance from data alone --- a known failure mode on small
molecular datasets where spatial symmetries are critical. Removing residual connections
destabilizes gradient flow in the 6-layer stack, causing training to converge to a
higher-loss basin.

\subsection{Moderate Contributors}
Attention-weighted aggregation (\(\Delta = +0.014\)) and layer normalization
(\(\Delta = +0.027\)) each provide tangible gains. The attention mechanism learns
chemically meaningful neighbor-weighting patterns (bond-order sensitivity), while
pre-normalization stabilizes training dynamics at larger learning rates.

\subsection{Diminishing Returns}
RBF distance encoding (\(\Delta = +0.011\)), edge feature updating (\(\Delta = +0.008\)),
and global context injection (\(\Delta = +0.006\)) show the smallest individual impacts.
This suggests redundancy: the model can partially recover distance information through
learned embeddings, and the 6-layer stack already captures long-range dependencies
without an explicit global token.

\subsection{Auxiliary Denoising}
The auxiliary coordinate-denoising loss provides a marginal gain (\(\Delta = +0.005\))
at negligible compute overhead. While statistically significant (\(p < 0.05\) on 5
seeds), it does not justify the added architectural complexity for this task.

\begin{findingbox}
  \textbf{\color{green}\faLightbulb\; Key finding:} The top 3 components --- residual
  connections, equivariant message-passing, and layer normalization --- collectively
  account for 67\% of the total degradation gap (\(0.076\) of \(0.166\)). A minimal
  viable architecture retaining only these three components achieves \metric\
  \(= 0.389\), within 25\% of the full model at 54\% of the parameter count.
\end{findingbox}

\subsection{Compute-Aware Architecture Design}
The compute-budget analysis (\S\,4) reveals that the most impactful components
(Residual, EquiMP, LN) do not carry disproportionate compute costs. The EquiMP
ablation actually \emph{reduces} parameter count by 18\%, suggesting that the
performance gain comes from inductive bias rather than increased capacity.

\section{Key Findings}
% ═══════════════════════════════════════════════════════════════

\begin{enumerate}[nosep,leftmargin=*,itemsep=2pt]
  \item \textbf{Equivariance is the strongest inductive bias.} Removing SE(3)-equivariant
        message-passing incurs the largest single-component degradation, confirming that
        rotational symmetry is not easily learned from data alone on \dataset.

  \item \textbf{Residual connections are non-negotiable for depth.} At 6 layers, the
        absence of skip connections destabilizes optimization and causes a \(\sim 9\%\)
        relative increase in \metric.

  \item \textbf{Three components dominate.} EquiMP, residuals, and layer normalization
        account for two-thirds of the total architecture contribution. A minimal model
        retaining only these achieves competitive performance at reduced parameter count.

  \item \textbf{Auxiliary losses provide diminishing returns.} The coordinate-denoising
        auxiliary objective yields a marginal gain not commensurate with added
        implementation complexity.

  \item \textbf{Compute cost does not correlate with impact.} The most impactful
        components (residuals, normalization) are parameter-free operations, while the
        least impactful (global context, edge updates) carry non-trivial parameter counts.
\end{enumerate}
