**Technical engineering design document (EDD) LaTeX template — document-control box, overview & goals, functional/non-functional requirement tables, a TikZ architecture diagram, interfaces, testing and risks.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/engineering-design-document)**, with real-time collaboration and one-second compiles.

![Engineering Design Document preview](preview.png)

## Features
- Document-control box (author, reviewers, status)
- Overview & goals
- Functional + non-functional requirement tables
- TikZ system-architecture diagram
- Interfaces, testing, risks

## Use it online (recommended)
Open **[Engineering Design Document on LetX »](https://letx.app/templates/assignments/engineering-design-document)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/engineering-design-document.git
cd engineering-design-document
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
