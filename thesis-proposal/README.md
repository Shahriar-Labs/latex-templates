**Academic thesis / dissertation proposal LaTeX template — formal title page (candidate, advisor, committee), then Abstract, Introduction, Research Questions, Related Work, Methodology, Preliminary Results, Timeline & References sections.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/thesis-proposal)**, with real-time collaboration and one-second compiles.

![Thesis Proposal preview](preview.png)

## Features
- Formal title page (candidate, advisor, committee)
- Abstract, Introduction, Research Questions
- Related Work, Methodology, Preliminary Results
- Milestone timeline table + references
- titlesec headings; realistic PhD sample

## Use it online (recommended)
Open **[Thesis Proposal on LetX »](https://letx.app/templates/academic-career/thesis-proposal)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/thesis-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
