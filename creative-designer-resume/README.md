**Bold creative resume LaTeX template for designers — full-width colored header band, two-column body, TikZ skill-level bars, and sections for profile, experience, skills, awards. One page, easily recolorable.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/cvs-resumes/creative-designer-resume)**, with real-time collaboration and one-second compiles.

![Creative Designer Resume preview](preview.png)

## Features
- Full-width colored header band with name, role & contact
- Two-column body; TikZ skill-level bars
- Profile, experience, skills, tools, awards sections
- Strong accent + neutral palette, easily recolorable
- One page, designed for creative roles

## Use it online (recommended)
Open **[Creative Designer Resume on LetX »](https://letx.app/templates/cvs-resumes/creative-designer-resume)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/creative-designer-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
