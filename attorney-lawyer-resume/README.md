## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/attorney-lawyer-resume.git
cd attorney-lawyer-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
