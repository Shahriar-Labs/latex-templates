## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/legal-pleading-paper.git
cd legal-pleading-paper
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
