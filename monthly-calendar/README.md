**Clean printable monthly calendar LaTeX template — a 7-column week grid with weekday headers, day numbers, roomy note cells, adjacent-month days greyed, and sample events. Easily change month and accent.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/calendars/monthly-calendar)**, with real-time collaboration and one-second compiles.

![Monthly Calendar preview](preview.png)

## Features
- 7-column week grid, weekday headers
- Day numbers + roomy note cells
- Adjacent-month days greyed
- Sample events; title + accent
- Easy month/colour change

## Use it online (recommended)
Open **[Monthly Calendar on LetX »](https://letx.app/templates/calendars/monthly-calendar)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/monthly-calendar.git
cd monthly-calendar
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
