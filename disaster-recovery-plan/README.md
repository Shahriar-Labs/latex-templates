## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/disaster-recovery-plan.git
cd disaster-recovery-plan
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
