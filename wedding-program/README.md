Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/wedding-program)**, with real-time collaboration and one-second compiles.

![Wedding Program preview](preview.png)

## Features
- Decorative header with couple names
- Order of Ceremony with dotted leaders
- Wedding Party section
- Thank-you note
- Elegant serif; one page

## Use it online (recommended)
Open **[Wedding Program on LetX »](https://letx.app/templates/miscellaneous/wedding-program)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/wedding-program.git
cd wedding-program
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
