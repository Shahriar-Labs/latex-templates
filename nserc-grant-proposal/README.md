## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/nserc-grant-proposal.git
cd nserc-grant-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
