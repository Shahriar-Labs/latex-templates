## Features
- Emerald & gold double border + arabesque header
- Photo placeholder; Bismillah header line
- Personal details incl. Sect/Maslak & Namaz
- Family (with Wali/Guardian) & Contact sections
- Perfectly aligned tables; one page

## Use it online (recommended)
Open **[Muslim Marriage Biodata on LetX »](https://letx.app/templates/miscellaneous/muslim-marriage-biodata)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/muslim-marriage-biodata.git
cd muslim-marriage-biodata
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
