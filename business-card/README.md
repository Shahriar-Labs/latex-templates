**Print-ready 10-up business card sheet LaTeX template — a 2x5 grid of standard 3.5x2in cards with monogram, contact icons, an accent stripe and dashed cut guides. Fully editable in TikZ.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/business-card)**, with real-time collaboration and one-second compiles.

![Business Card Sheet preview](preview.png)

## Features
- Print-ready 10-up (2x5) sheet, standard 3.5x2in
- Monogram, contact icons, accent stripe
- Address strip + decorative dots
- Dashed cut guides at intersections
- Fully editable TikZ card macro

## Use it online (recommended)
Open **[Business Card Sheet on LetX »](https://letx.app/templates/miscellaneous/business-card)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/business-card
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
