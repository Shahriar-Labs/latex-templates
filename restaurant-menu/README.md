**Elegant fine-dining restaurant menu LaTeX template — decorative header, sections (starters, mains, desserts, beverages) with dish names, dotted price leaders and italic descriptions. Classic serif typography.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/restaurant-menu)**, with real-time collaboration and one-second compiles.

![Restaurant Menu preview](preview.png)

## Features
- Decorative header + ornament rule
- Sections: starters, mains, desserts, beverages
- Dish names with dotted price leaders
- Italic dish descriptions
- Classic serif; upscale sample

## Use it online (recommended)
Open **[Restaurant Menu on LetX »](https://letx.app/templates/miscellaneous/restaurant-menu)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/restaurant-menu.git
cd restaurant-menu
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
