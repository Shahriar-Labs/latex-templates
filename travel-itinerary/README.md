# Travel Itinerary — Free LaTeX Template

**Modern day-by-day travel itinerary LaTeX template — colored header band, flight/hotel/budget summary with icons, and per-day activity cards with times, locations and notes. Includes a realistic 3-day sample trip.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/miscellaneous/travel-itinerary)**, with real-time collaboration and one-second compiles.

![Travel Itinerary preview](preview.png)

## Features
- Colored header band + trip summary block
- Flight / hotel / budget with FontAwesome icons
- Per-day activity cards (time, place, notes)
- Tasteful travel palette; realistic 3-day sample
- Clean, gift-ready layout

## Use it online (recommended)
Open **[Travel Itinerary on LetX »](https://letx.app/templates/miscellaneous/travel-itinerary)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/travel-itinerary.git
cd travel-itinerary
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
