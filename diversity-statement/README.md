Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/diversity-statement)**, with real-time collaboration and one-second compiles.

![Diversity Statement preview](preview.png)

## Features
- Title block + clean academic styling
- My Commitment to Diversity
- Past Contributions & Experience (bulleted)
- Inclusive Teaching & Future Plans
- Thoughtful, realistic sample content

## Use it online (recommended)
Open **[Diversity Statement on LetX »](https://letx.app/templates/academic-career/diversity-statement)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/diversity-statement
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
