Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/lab-documents/geology-field-report)**, with real-time collaboration and one-second compiles.

## Features
- Site info box + abstract
- TikZ stratigraphic column
- Lithology + sample-catalog tables
- Structural observations + interpretation
- Realistic field-geology sample

## Use it online (recommended)
Open **[Geology Field Report on LetX »](https://letx.app/templates/lab-documents/geology-field-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/geology-field-report.git
cd geology-field-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
