**Clean academic research statement LaTeX template for faculty job applications — title block, boxed research vision, and Overview, Past & Current Research, Future Directions, Broader Impacts & Publications sections.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/academic-career/research-statement)**, with real-time collaboration and one-second compiles.

![Research Statement preview](preview.png)

## Features
- Refined title block + boxed research vision
- titlesec section headings with accent rules
- Overview, Past/Current Research, Future Directions
- Broader Impacts + selected publications
- Realistic CS sample content; ~2 pages

## Use it online (recommended)
Open **[Research Statement on LetX »](https://letx.app/templates/academic-career/research-statement)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/research-statement.git
cd research-statement
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
