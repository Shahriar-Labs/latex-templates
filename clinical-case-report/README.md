**Medical clinical case report LaTeX template — structured abstract (background/case/conclusions), case presentation with a vitals/labs table, investigations, differential diagnosis, management and discussion. Journal style.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/clinical-case-report)**, with real-time collaboration and one-second compiles.

![Clinical Case Report preview](preview.png)

## Features
- Structured abstract (background/case/conclusions)
- Case presentation + vitals/labs table
- Investigations + differential diagnosis
- Management & outcome, discussion
- Journal style; anonymized sample

## Use it online (recommended)
Open **[Clinical Case Report on LetX »](https://letx.app/templates/journal-articles/clinical-case-report)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/clinical-case-report
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
