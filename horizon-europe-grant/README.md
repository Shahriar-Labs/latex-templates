## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/horizon-europe-grant.git
cd horizon-europe-grant
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
