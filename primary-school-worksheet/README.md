## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/primary-school-worksheet.git
cd primary-school-worksheet
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
