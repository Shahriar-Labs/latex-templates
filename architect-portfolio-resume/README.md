## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/architect-portfolio-resume.git
cd architect-portfolio-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
