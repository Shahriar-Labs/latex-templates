**Telescope observing-time proposal template (Gemini / ESO / NOIRLab TAC style) — styled proposal header, scientific & technical justification, and a professional target-list table.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/observing-time-proposal)**, with real-time collaboration and one-second compiles.

![Telescope Observing Proposal preview](preview.png)

## Features
- Observatory **TAC proposal** layout (Gemini / ESO / NOIRLab style)
- Styled header: PI, Co-Is, instrument, nights, and semester
- Scientific + Technical Justification sections with a pgfplots figure
- Professional **target-list table** (RA/Dec/mag/exposure/priority)
- Realistic high-redshift Lyman-alpha spectroscopy sample content

## Use it online (recommended)
Open **[Telescope Observing Proposal on LetX »](https://letx.app/templates/grants/observing-time-proposal)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/observing-time-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
