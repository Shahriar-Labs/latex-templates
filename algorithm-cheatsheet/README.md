**Dense, color-coded algorithms & data-structures cheatsheet LaTeX template — multi-column tcolorbox topic cards, Big-O complexity tables, and syntax-highlighted code for sorting, searching, graphs & dynamic programming.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/assignments/algorithm-cheatsheet)**, with real-time collaboration and one-second compiles.

![Algorithm Cheatsheet preview](preview.png)

## Features
- Dense multi-column, color-coded tcolorbox cards
- Big-O complexity tables (booktabs)
- Syntax-highlighted code (sorting, search, BFS, quicksort)
- Data-structure operation complexities
- Genuinely useful exam/interview reference

## Use it online (recommended)
Open **[Algorithm Cheatsheet on LetX »](https://letx.app/templates/assignments/algorithm-cheatsheet)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/algorithm-cheatsheet
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
