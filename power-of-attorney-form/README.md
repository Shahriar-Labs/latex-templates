## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/power-of-attorney-form.git
cd power-of-attorney-form
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
