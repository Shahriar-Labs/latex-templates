## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/ip-assignment-agreement.git
cd ip-assignment-agreement
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
