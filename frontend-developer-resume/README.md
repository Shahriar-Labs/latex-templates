## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/frontend-developer-resume.git
cd frontend-developer-resume
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
