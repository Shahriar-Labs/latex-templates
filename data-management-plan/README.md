Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/data-management-plan)**, with real-time collaboration and one-second compiles.

![Data Management Plan preview](preview.png)

## Features
- NSF/NIH DMP structure, numbered sections
- Data types & formats, metadata standards
- Storage, sharing, preservation policy
- Data-repository summary table
- Realistic research sample

## Use it online (recommended)
Open **[Data Management Plan on LetX »](https://letx.app/templates/grants/data-management-plan)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/data-management-plan.git
cd data-management-plan
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
