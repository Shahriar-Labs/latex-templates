## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/ui-indonesia-thesis.git
cd ui-indonesia-thesis
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
