**NASA ROSES / NSPIRES research grant proposal template — cover block with program element, technical merit, work plan, budget & timeline tables, and references. For NASA astrophysics proposals.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/grants/nasa-roses-proposal)**, with real-time collaboration and one-second compiles.

![NASA ROSES Grant Proposal preview](preview.png)

## Features
- **NASA ROSES / NSPIRES** proposal structure with program-element cover block
- Scientific/Technical Merit, Background, Methodology, and Relevance sections
- Work-plan **timeline table** and a **budget-by-year table** (`booktabs`)
- A self-contained pgfplots figure and an inline bibliography
- Realistic AGN-feedback astrophysics research content

## Use it online (recommended)
Open **[NASA ROSES Grant Proposal on LetX »](https://letx.app/templates/grants/nasa-roses-proposal)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/latex-templates.git
cd latex-templates/nasa-roses-proposal
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
