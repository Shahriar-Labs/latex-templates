\section{General Description}
% =============================================================================
The ADX2050 is a dual-channel, precision operational amplifier optimized for
low-noise, low-offset signal conditioning in industrial and instrumentation
applications. Fabricated on Apex Semiconductor's third-generation complementary
bipolar process, the ADX2050 combines rail-to-rail output swing with a wide
3\,V to 32\,V single-supply operating range, enabling direct interfacing with
sensor bridges, thermocouples, and precision data-acquisition front ends.

\medskip
Each amplifier channel is unity-gain stable and offers a 10\,MHz
gain-bandwidth product with a 5.5\,V/\textmu s slew rate, making the device
well suited to active filtering and moderate-speed signal-processing tasks.
The ADX2050 is offered in an 8-pin SOIC package and is fully specified over
the $-40\,^{\circ}$C to $+125\,^{\circ}$C industrial temperature range.

% =============================================================================
\section{Features}
% =============================================================================
\begin{itemize}
  \item Ultra-low input offset voltage: 0.3\,mV (typ), 1.2\,mV (max)
  \item Low input bias current: 45\,nA (typ)
  \item Wide supply range: 3\,V to 32\,V single-supply, $\pm$1.5\,V to $\pm$16\,V dual-supply
  \item Rail-to-rail output swing
  \item Unity-gain stable, 10\,MHz gain-bandwidth product
  \item Low voltage noise density: 8\,nV/$\sqrt{\text{Hz}}$ at 1\,kHz
  \item Industrial temperature range: $-40\,^{\circ}$C to $+125\,^{\circ}$C
  \item ESD protection: 2\,kV (HBM), 1\,kV (CDM)
  \item Available in 8-pin SOIC package
\end{itemize}

% =============================================================================
\section{Applications}
% =============================================================================
\begin{itemize}
  \item Precision instrumentation amplifiers
  \item Sensor and bridge signal conditioning
  \item Battery-powered and portable medical instrumentation
  \item Audio pre-amplification
  \item Industrial process-control loops (4--20\,mA)
  \item Active filters and signal buffering
\end{itemize}
