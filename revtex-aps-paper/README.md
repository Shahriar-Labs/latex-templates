**REVTeX 4.2 physics paper template (APS / Physical Review) — two-column, author affiliations, displayed equations, a pgfplots figure, a results table, and a bibliography. Compile-tested.**

Edit and compile this template instantly in your browser — no LaTeX install — at **[letx.app](https://letx.app/templates/journal-articles/revtex-aps-paper)**, with real-time collaboration and one-second compiles.

![REVTeX 4.2 Physics Paper preview](preview.png)

## Features
- APS **REVTeX 4.2** class — the standard for *Physical Review* (PRD/PRL/PRX) submissions
- Two-column layout with `superscriptaddress` author affiliations
- Displayed equations, a self-contained **pgfplots** figure, and a results table
- Inline bibliography with `\cite` — no external `.bib` needed
- Realistic dark-matter / gravitational-wave sample content

## Use it online (recommended)
Open **[REVTeX 4.2 Physics Paper on LetX »](https://letx.app/templates/journal-articles/revtex-aps-paper)** and click *Open as Template* — it compiles in ~1 second, in your browser, free.

## <a name="compile"></a>Compile locally
```bash
git clone https://github.com/Shahriar-Labs/revtex-aps-paper.git
cd revtex-aps-paper
latexmk -pdf main.tex
```
Compiler: **pdflatex** (see `metadata.json`).
